端口映射 vs 宿主机映射

| 项目         | 端口映射                         | 宿主机映射                       |
| ------------ | -------------------------------- | -------------------------------- |
| **本质**     | 网络通信技术                     | 文件系统共享技术                 |
| **作用**     | 让外部能访问内部服务             | 让宿主机和容器共享文件           |
| **使用场景** | 外部访问内网服务                 | 本地修改代码/配置                |
| **配置位置** | 路由器/防火墙/Docker 端口设置    | Docker Compose 文件/volumes      |
| **形象比喻** | 电话转接服务（把电话转到特定人） | 文件共享盘（把文件放在共享盘里） |

**端口映射是将一个端口的请求转发到另一个端口的技术**，主要用于让外部网络访问内部网络中的服务。
